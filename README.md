# Timber Fuel — Drinks Done Bold

Static marketing/menu site for **Timber Fuel**, a coffee and energy-refresher stand in Cass Lake, Minnesota.

> Fuel for the Northwoods · Northwoods Crafted · Est. 2024

## Stack
Single-file static site. Plain HTML + CSS + a little vanilla JS. No build step.

- `index.html` — the whole site (hero, sticky tab nav, Hot / Cold / Energy menu sections, footer)
- `assets/` — logo, illustrated hero background, drink photos, and section icons

Fonts: Anton (display), Oswald (subheads/labels), Barlow Condensed (body) via Google Fonts.

## Sections
- **Hero** — illustrated Northwoods sunset, "Hot · Cold · Energy / Drinks Done Bold", hours strip
- **Cold Drinks** — signature cold-brew combos + interactive "Prefer Your Own Mix?" builder
- **Hot Drinks** — drink list + customize panel (sizes + flavor add-ons)
- **Energy Refreshers** — four color-coded flavors + Spicy Mango spotlight
